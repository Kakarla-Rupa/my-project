# 🚀 Friend Project (RAG / AI Project)

## 📌 Overview
This project is an AI-based application that uses **RAG (Retrieval-Augmented Generation)** to answer user queries based on documents.

It combines:
- FastAPI (backend)
- Streamlit (frontend)
- LLM (Ollama / local models)
- Document processing (PDF/TXT)

---

## 🛠️ Tech Stack

- 🐍 Python
- ⚡ FastAPI
- 🎨 Streamlit
- 🤖 LLM (Ollama)
- 📚 LangChain (optional)
- 📄 PDF Processing
